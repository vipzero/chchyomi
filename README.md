# chchyomi

> 5ch 読み上げ CLI

## Usage
    $ chchyomi [thread URL]

## Options
    --char, -c talk char length (default 40)

## Examples
    # スレの読み上げ 
    $ chchyomi https://hebi.5ch.net/test/read.cgi/news4vip/1234567890/
    # 
    $ chchyomi https://hebi.5ch.net/test/read.cgi/news4vip/1234567890/ -c 20
